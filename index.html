About

Machine learning and deep learning for medical imaging and intelligent healthcare.

My research interests revolve around developing advanced machine learning and deep learning techniques for medical image analysis and healthcare diagnostics. I specialize in designing computer vision models for disease detection, classification, and image enhancement across multiple imaging modalities.

I am particularly interested in convolutional neural networks, ensemble learning, transfer learning, and image preprocessing for building robust pipelines that enhance diagnostic accuracy and reliability.

Selected Research Projects

Medical Imaging · Deep Learning

Automated Diabetic Retinopathy Detection Using Deep Learning Models

Developing an ensemble-based and hybrid deep learning framework for detecting diabetic retinopathy from retinal fundus images. The pipeline incorporates CLAHE, data augmentation, ensemble CNN architectures, and multiple evaluation metrics to enhance diagnostic performance.

Status: Manuscript under review in Healthcare Analytics (Q1).

Behavior Analysis · ML

Autism Spectrum Disorder Detection Using Machine Learning Approaches

Early detection of Autism Spectrum Disorder (ASD) by analyzing behavioral and social communication characteristics using machine learning techniques. Evaluates 10 key features (AQ1–AQ10) across age-based datasets and systematically tests 271 classifiers, identifying nine high-performing models with accuracy between 97.2% and 99.1%.

NLP · AI Safety

AI Generated Text and Code Detection Using ML/DL Models

Designing models capable of distinguishing AI-generated text and source code from human-written content using CNNs, GRUs, hybrid LSTM-CNN, and transformer-based architectures. Performance is evaluated on a comprehensive dataset of LLM-produced and human-written essays, stories, and poetry using standard classification metrics.
